# Azure Resource Group with Terraform (Module-based architecture)

## 🚀 Project Overview

This project provisions an Azure Resource Group using **Terraform with a reusable module-based architecture.**
The goal is to follow a **clean, scalable, and professional Terraform structure**.

## 🏗️ Architecture

- Root module handles input variables and module calls
- Child module handles actual Resource Group creation
- Outputs are exposed back to root module

## ⚙️ Features

- Modular Terraform design
- Azure Resource Group deployment
- Environment-based configuration support
- Clean separation of root and module logic


## 🌍 Environment

- Azure Region: **Japan East**
- Resource Type: **Resource Group**
